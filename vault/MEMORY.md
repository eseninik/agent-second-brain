## Patterns & Learnings

### What Works
- Deep work sessions утром
- Voice notes для быстрого захвата мыслей
- Weekly reviews для рефлексии

### What Doesn't Work
- Абстрактные задачи без next action
- Слишком длинные заголовки
- Работа без clear priorities
